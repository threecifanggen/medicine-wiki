# 槌状趾

## 症状

远端趾间关节屈曲畸形并伴有疼痛。**第二趾**最常受累。此外的症状包括：

1. 主述没有合适的鞋穿
2. 走路时加重，休息、热敷缓解
3. 疼痛的特点是持续性酸痛
4. 部分会听到关节弹响、查体时出现捻发音。
5. 还有关节畸形。
