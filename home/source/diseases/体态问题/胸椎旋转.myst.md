# 胸椎旋转

## 介绍和原因

可以看到身体一侧较另一侧更靠近检查者身体。一般由习惯性动作导致。

## 向右前/顺时针旋转时的肌肉

1. 缩短的肌肉
   1. 右侧的深部胸椎旋转肌
   2. 右腹内斜肌
   3. 左腹外斜肌
   4. 左侧腰肌
   5. 左侧腰椎竖脊肌
   6. 让颈部向左侧的肌肉
2. 延长的肌肉
   1. 左侧的深部胸椎旋转肌
   2. 左腹内斜肌
   3. 右腹外斜肌
   4. 右侧腰肌
   5. 右侧腰椎竖脊肌
   6. 让颈部转向右侧的肌肉

## 处理

### 治疗师处理

1. 了解原因
2. 帮助病人朝相反的方向拉伸
3. 一些整体筋膜拉伸
4. 按摩缩短的组织
