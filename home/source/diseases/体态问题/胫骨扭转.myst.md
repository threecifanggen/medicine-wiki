# 胫骨扭转

## 症状和原因

1. 日本人跪姿容易产生
2. 一般左右脚有差异

## 影响

1. 高尔夫等会加重
2. 改变半月板在髌骨上滑动的相关位置
3. 影响足和踝关节
4. 下肢会不平衡

## 处理

### 治疗师处理

1. 改变不对称姿势
2. 避免姿势
3. 贴扎
4. 处理肌肉不平衡

### 自我处理

1. 避免扭转姿势
2. 减少旋转多的运动
3. 使用护具
