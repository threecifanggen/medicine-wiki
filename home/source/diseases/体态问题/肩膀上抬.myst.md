# 肩膀上抬

## 原因和症状

肩胛骨一般落在第二到第七肋骨。当拉到上方，就被称为上抬。

```{figure} /_static/img/2022-02-03-12-55-25.png
肩膀上抬案例
```

## 影响

1. 影响到斜方肌、前锯肌、菱形肌和提肩胛肌。
2. 上抬颈部足趾压迫，对侧延长。造成颈部影响

## 处理

### 治疗师处理

1. 查看是不是单侧肩膀受力：靠车窗、撑头
2. 帮助病人下降
3. 被动伸展肩胛提肌和斜方肌


### 病人处理

1. 练习肩胛下降
2. 主动伸展缩短的肌群
