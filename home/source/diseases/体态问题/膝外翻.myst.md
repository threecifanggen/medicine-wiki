# 膝外翻

## 症状和原因

又叫「X型腿」。通常伴随长短腿。

## 影响

1. 内侧韧带拉力增大
2. 股薄肌、半腱肌、缝匠肌会被延长而紧绷，阔筋膜张肌、髂胫束和小腿外侧被压迫

## 处理

### 治疗师处理

1. 转入足科
2. 拉伸和按摩髂胫束
3. 股薄肌按摩
4. 贴扎膝关节内侧

### 自我处理

1. 避免膝外翻懒人站姿
2. 主动伸展髋关节内收和外展肌群
3. 滚筒处理髂胫束
4. 避免高冲击运动
5. 考虑护膝
